Selectorsالمُحَدِّدات

تأتي CSS بمجموعة كبيرة من ال selectors (المُحدِّدات)، ويمكن دمج ومزج هذه المُحدِّدات بطرق مثيرة للاهتمام.

ومن أبسط هذه ال selectors أسماء العناصر في HTML أو ال Class Name الخاص بهم: 

/* Turn all links red! */
a {
  color: red;
 }

/*
  Remove the underline from all elements that
  have been given a class of `navigation-link`
*/
.navigation-link {
  text-decoration: none;
 }

في المثال السابق، استخدمنا إسم العنصر a لتحديد أي عنصر في صفحتنا من النوع anchor، ثم طبقنا عليه الأمر وهو التلوين باللون الأحمر.

ثم قمنا بتحديد أي عنصر يحمل class بقيمة navigation-link وطبقنا عليه الأمر، وهو إزالة الخط السفلي.
