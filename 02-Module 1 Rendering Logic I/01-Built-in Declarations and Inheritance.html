Rendering Logic I

الهدف الأساسي من CSS هو منحك القدرة على التحكم في كل من مظهر وترتيب محتوى صفحتك.

عندما نبدأ في تصميم وترتيب صفحتنا، الصفحة لا تكون فارغة تمامًا؛ لأن ال HTML Tags تحتوي على بعض الـ Default Styles "أي بعض الـ styles التي تكون مُضمنة في عناصر HTML افتراضيًا". على سبيل المثال، الـعنصر <a> في HTML يحتوي على الـ styles الآتية افتراضياً: 

a {
  color: -webkit-link;
  cursor: pointer;
  text-decoration: underline;
 }

تلك الـ Styles هي جزء مما يسمى بـ User-agent Stylesheet. كل متصفح يحتوي على Stylesheet خاص به يشمل مجموعة من الـ Base Styles مثل هذه. على سبيل المثال: من الممكن أن تختلف الـ Default Styles الخاصة بعنصر ما في متصفح Chrome عن الـ Default Styles الخاصة بنفس العنصر في متصفح Firefox.

إذا كنت قد استخدمت CSS Reset من قبل، فهو يهدف إلى "موازنة" User-agent Stylesheets عبر المتصفحات المختلفة، ويضمن أن كل متصفح يبدأ بنفس مجموعة الـ Core Styles. "وذلك لجعل code الـ CSS الخاص بنا يعمل على كل المتصفحات بنفس الطريقة، وإلا ستحدث بعض المشاكل بسبب اختلاف ال Default Styles من للعناصر في HTML متصفح إلى آخر."
